# 27.2 WPS Office（Linux 版）

WPS Office 没有原生 FreeBSD 版本，且 Ports 中的 WPS 已无人维护，需通过自行构建的 Linux 兼容层安装。本节基于 Rocky Linux 兼容层给出安装与配置步骤。

> **警告**

> 请勿使用 Ports 中的 WPS Office，因为该 Port 已无人维护。请自行构建兼容层后安装使用。

## 基于 Rocky Linux 兼容层（FreeBSD Port）

> **注意**
>
> 请先参考本书其他章节完成 Rocky Linux 兼容层（FreeBSD Port）的安装。

### 安装 RPM 工具

使用 pkg 安装：

```sh
# pkg install rpm4
```

或者使用 Ports 方式安装：

```sh
# cd /usr/ports/archivers/rpm4/
# make install clean
```

### 下载 WPS Office

WPS Office 官方下载地址：[WPS Office for Linux](https://linux.wps.cn/)。该页面提供 Linux 版 WPS Office 的官方下载渠道。

请自行获取有效链接。

> **技巧**
>
> 该链接存在问题，目前无法通过 fetch 或 wget 下载。如有解决方案，请提交 PR 或 issue。

### 安装 WPS Office

切换到兼容层路径：

```sh
# cd /compat/linux/
```

安装 WPS Office：

```sh
# rpm2cpio < /home/ykla/Downloads/wps-office-12.1.0.17900-1.x86_64.rpm  | cpio -id
```

请将路径替换为本地下载文件的实际路径。

WPS Office 文件结构：

```sh
/compat/linux/
└── opt/
    └── kingsoft/
        └── wps-office/
            └── office6/
                └── wps # WPS 可执行文件
```

### 运行 WPS Office

使用普通用户权限在 Linux 兼容环境中启动 WPS Office：

```bash
$ export QT_IM_MODULE=fcitx
$ /compat/linux/opt/kingsoft/wps-office/office6/wps
```

## 基于 Arch Linux 兼容层

```sh
# chroot /compat/arch/ /bin/bash # 进入 Arch 兼容层
# passwd # 为 Arch 的 root 设置密码
# passwd test # 为 Arch 的 test 用户设置密码，该用户已由脚本创建
```

## 基于 Ubuntu 的兼容层

进入 Ubuntu 兼容层：

```sh
# chroot /compat/ubuntu/ /bin/bash
```

安装依赖包：

```sh
# apt install bsdmainutils xdg-utils libxslt1.1 libqt5gui5 xcb
```

下载 WPS Office Debian 安装包：

```sh
# wget https://wps-linux-personal.wpscdn.cn/wps/download/ep/Linux2019/11698/wps-office_11.1.0.11698_amd64.deb
```

调用包管理器安装 WPS Office：

```sh
# apt install ./wps-office_11.1.0.11698_amd64.deb
```

安装完毕。

>
> 写作本节时，Fcitx5 输入法未生效，功能待后续测试。如有解决方法，请提供参考。

## 故障排除与未竟事宜

### 无法启动

查看 `/usr/lib/office6/wps` 可执行文件的动态库依赖：

```sh
# ldd /usr/lib/office6/wps
```

根据 `ldd` 输出补充缺失的库。

### 需要 root 权限才能启动

有待解决。

### 由于缺少 Bash 导致 WPS 无法启动

由于 WPS 的启动文件需要调用 Bash Shell，因此安装 Bash 后即可正常启动。

- 使用 pkg 安装：

```sh
# pkg install bash
```

- 或者使用 Ports 安装：

```sh
# cd /usr/ports/shells/bash/
# make install clean
```
