# 8.3 Mihomo（原 Clash）

mihomo 在 [ports](https://www.freshports.org/net/mihomo) 中就有提供。

## 安装 mihomo

- 使用 pkg 安装：

```sh
# pkg install net/mihomo
```

- 或者使用 Ports 安装：

```
# cd /usr/ports/net/mihomo/ 
# make install clean
```

如果觉得麻烦可以在 Linux 兼容层中使用，FreeBSD 网络一样可受其控制。
