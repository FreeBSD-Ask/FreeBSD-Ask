# 10.7 文档查看器

自 UNIX 问世以来，许多新的文档格式逐渐流行，如 PDF、DJVU、MOBI、AZW3、EPUB 等，而这些格式所需的查看器在基本系统中或许并不存在。本节介绍如何安装文档查看器。

## Calibre 文档管理（PDF、EPUB、MOBI、AZW3 等格式）

Calibre 是电子书管理工具，支持多种电子书格式的阅读、转换和组织，支持自定义 CSS 样式。

使用 pkg 安装：

```sh
# pkg install calibre
```

使用 Ports 安装：

```sh
# cd /usr/ports/deskutils/calibre/
# make install clean
```

## Okular

Okular 是一款通用文档查看器，属于 KDE Plasma 项目的一部分。Okular 结合了出色的功能，支持多种文档格式，如 PDF、Postscript、DjVu、CHM、XPS、ePub 等。

使用 pkg 安装 Okular：

```sh
# pkg install okular
```

## Evince

Evince 是一款支持多种文档格式（包括 PDF 和 Postscript）的文档查看器，属于 GNOME 项目的一部分。Evince 的目标是用一个简单的应用程序替代像 ggv 和 gpdf 这样的文档查看器。

使用 pkg 安装 Evince：

```sh
# pkg install evince
```

## ePDFView

ePDFView 是一款轻量级的 PDF 文档查看器，仅使用 Gtk+ 和 Poppler 库。ePDFView 的目标是提供一个简单的 PDF 文档查看器，类似于 Evince，但不使用 GNOME 库。

使用 pkg 安装 ePDFView：

```sh
# pkg install epdfview
```

## Xpdf

对于喜欢小巧 FreeBSD PDF 查看器的用户，Xpdf 提供了一款轻量高效的查看器，所需资源较少。它使用标准的 X 字体，不需要额外的工具包。

使用 pkg 安装 Xpdf：

```sh
# pkg install xpdf
```

## Zathura

Zathura 是一款高度可定制且功能强大的文档查看器。它提供简约、节省空间的界面，主要专注于键盘交互，使用起来非常便捷。

使用 pkg 安装支持 PDF 的 Zathura：

```sh
# pkg install zathura zathura-pdf-mupdf
```

此外，还可以安装 graphics/zathura-pdf-poppler 提供替代的 PDF 支持，

[graphics/zathura-ps](https://cgit.freebsd.org/ports/tree/graphics/zathura-ps/) 提供 PostScript 支持

[graphics/zathura-djvu](https://cgit.freebsd.org/ports/tree/graphics/zathura-djvu/) 提供 DjVu 支持

[graphics/zathura-cb](https://cgit.freebsd.org/ports/tree/graphics/zathura-cb/) 提供漫画书支持
