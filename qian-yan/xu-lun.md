# 绪论

绪论提供本书的整体概览，包括本书定位、对读者的要求、组织结构、书中命令及符号含义。

## 本书定位

本书涵盖 FreeBSD 15.0-RELEASE 与 14.4-RELEASE 的安装与日常使用，并尽可能向下兼容较早版本，同时包含部分 16.0-CURRENT 的内容。

本书主要面向 amd64（x86-64）与 arm64（AArch64）架构，并尽可能支持其他硬件架构。

本书中的测试环境为 Windows 11，并保持系统更新。

## 对读者的要求

本书以高等院校计算机科学与技术专业一般本科毕业生的及格或以上水平作为难度基准。读者应具备以下基础知识：

- 操作系统基本概念（进程、内存管理、文件系统等）
- 命令行操作经验
- 计算机网络基础知识
- 英文阅读能力（用于阅读官方文档与 man 页面）

## 本书的组织结构

**第 1 章 FreeBSD 初见**：介绍 BSD UNIX 的历史，涵盖自由软件运动与开源运动的发展脉络。

**第 2 章 FreeBSD 安装基础**：指导使用 FreeBSD 内置安装工具 `bsdinstall` 完成系统安装，包括磁盘分区、用户创建、网络配置等步骤。

**第 3 章 FreeBSD 高级安装**：介绍在虚拟机、云服务器及特殊硬件等平台上安装 FreeBSD 的方法。

**第 4 章 FreeBSD 基础**：涵盖基本命令、文件系统权限、Shell 配置等内容。

**第 5 章 系统管理**：讲解启动引导、服务管理、用户分级、SSH、远程桌面、定时任务、系统配置以及系统恢复等管理功能。

**第 6 章 网络管理**：介绍网络基础配置、TCP/IP 协议栈、无线网络（Wi-Fi）配置和 USB 网络共享。

**第 7 章 网络代理工具**：介绍系统代理配置及 V2Ray、Mihomo 等代理工具的使用。

**第 8 章 存储与文件系统管理**：介绍 UFS 文件系统、自动挂载、Windows 和 Linux 文件系统的使用以及 Swap 分区配置。

**第 9 章 ZFS 文件系统**：涵盖 ZFS 概述、快照与还原、磁盘扩容、数据压缩、启动环境、存储池更新及委托管理。

**第 10 章 软件管理和系统升级**：介绍使用 pkg 安装二进制包、使用 Ports 编译安装软件以及使用 freebsd-update 进行系统更新。

**第 11 章 桌面环境**：介绍在 FreeBSD 上运行的桌面环境与窗口管理器，包括 GNOME、KDE、Xfce 等。

**第 12 章 人工智能（AI）**：介绍在 FreeBSD 上运行大语言模型及相关工具的方法。

**第 13 章 本地化与中文支持**：介绍中文输入法框架及系统字体配置。

**第 14 章 多媒体和外部设备**：介绍 Web 浏览器、音视频播放与处理，以及蓝牙、打印机等外部设备的配置。

**第 15 章 Jail 容器管理**：介绍 FreeBSD 原生轻量级虚拟化技术 jail 的创建、配置与网络设置。

**第 16 章 虚拟化与容器管理**：介绍 bhyve 虚拟机管理程序及其他容器化方案。

**第 17 章 防火墙**：介绍 FreeBSD 内置防火墙系统，包括 PF、IPFW 和 IPFilter 的配置。

**第 18 章 服务器**：介绍 FTP 服务器、Samba 文件共享、NFS 服务器、零配置网络（mDNS/DNS-SD）等服务的搭建。

**第 19 章 Web 服务器与监控系统**：介绍 Apache、Nginx、Caddy、Prometheus、Grafana、Nextcloud 等 Web 服务与监控平台的部署。

**第 20 章 嵌入式平台**：介绍在树莓派等 ARM 设备以及 RISC-V 和 x86 嵌入式开发板上运行 FreeBSD 的方法。

**第 21 章 数据库管理**：介绍 PostgreSQL、MySQL、MongoDB 等数据库的安装与配置。

**第 22 章 游戏、科学计算和专业工具**：介绍游戏引擎、科学计算及专业软件在 FreeBSD 上的使用方法。

**第 23 章 Linux 二进制兼容层**：介绍在 FreeBSD 上运行各类 Linux 发行版的方法。

**第 24 章 开发环境**：介绍 C/C++、Java、Python、Rust 等语言开发环境的搭建，以及 DTrace 动态追踪工具的使用。

**第 25 章 社区参与**：介绍 Bug 报告流程、邮件列表订阅及 FreeBSD 开发参与方式。

**第 26 章 基于 Linux 兼容层的应用**：介绍基于 Linux 兼容层运行 QQ、WPS Office、微信及 Wine 等应用。

**第 27 章 FreeBSD 内核架构**：介绍 FreeBSD 内核选项与定制内核的构建。

**附录 I 工具与资源**：包括硬件挑选建议及开源常识。

**附录 II 文学故事**：文学性内容。

**附录 III 参考文献与术语表**：列出全书核心参考文献和术语表。

### 权限标识

`#` 表示在 `root` 权限下的操作，通常通过 `su`、`sudo` 或 `doas` 获得。

`$`、`%` 表示普通用户权限（注：`%` 在 FreeBSD 文档中不常用作标准提示符，通常使用 `$`）。

## pkg 与 ports

FreeBSD 提供两种软件安装方式：pkg（二进制包管理）和 Ports（源代码编译）。这两种方式各有优势，可以根据实际需求选择使用，也可以混合使用。

pkg 是 FreeBSD 的二进制包管理器，通过预编译的软件包提供快速、便捷的安装方式，适合大多数用户的日常使用。Ports 则提供了从源代码编译安装的方式，允许用户根据需要自定义配置选项，适合需要定制化或优化的场景。

>**请注意**
>
> Ports 通常对应 HEAD 分支，建议 pkg 与 Ports 保持在同一主线上，即均选择 `latest`。亦可自行使用 pkg 对应的 Ports 季度分支，如 `2025Q1`。

如果要安装软件 `yyy`，`yyy` 在 Ports 里是 `xxx/yyy`，那么路径即为 `/usr/ports/xxx/yyy`。

首先可通过 pkg 安装二进制软件包，与绝大多数 Linux 发行版的用法一致：

```sh
# pkg install yyy
```

亦可采用以下方式：

```sh
# pkg install xxx/yyy
```

或采用简写形式：

```sh
# pkg ins yyy
```

还可通过 Ports 编译安装：

```sh
# cd /usr/ports/xxx/yyy
# make install clean
```

系统将陆续弹出配置窗口供用户选择。若采用默认选项，可执行以下命令：

```sh
# cd /usr/ports/xxx/yyy
# make BATCH=yes install clean
```

若需一次性完成所有依赖配置：

```sh
# cd /usr/ports/xxx/yyy
# make config-recursive # 将递归询问直至所有依赖配置完毕
# make install clean
```
