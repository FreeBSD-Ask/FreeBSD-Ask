# FreeBSD 从入门到追忆（第三版：草稿）

FreeBSD 是真正自由（Liberty）的**操作系统**，在这波谲云诡的世界中仍然坚守 BSD UNIX 哲学——恪守古老的法则，追寻真正的自由。

## 关于

### 项目名称为何是《FreeBSD 从入门到追忆》

此标题意在对传统技术类书籍《XX 从入门到精通》这一命名方式进行解构与反叛。“且有大觉而后知此其大梦也”：一切执着与努力，最终不过化为个人心底那一抹冷色调地追忆。

世界也许不是真实的，物质也许是虚妄的，FreeBSD 可能根本不存在——但是我们的追忆却是那么清晰那么遗憾。
