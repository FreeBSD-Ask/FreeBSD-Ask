# 14.3 USB 网络共享（USB tethering）

## USB 网络共享技术概述

USB 网络共享（USB tethering）是一种通过通用串行总线（Universal Serial Bus，USB）物理层将移动设备（如智能手机、平板电脑）的广域网（WAN）连接共享给计算机系统的技术。该技术实现了移动设备网络连接的复用，为缺乏独立网络接入的计算设备提供了临时网络连接方案。

## 测试验证与兼容性说明

本章介绍 FreeBSD 操作系统中 USB 网络共享的配置方法，该教程已在红米手机（红米 Note 12 5G）上完成测试验证，从技术原理上可同时支持安卓（Android）和苹果（iOS）设备。

### WiFi 共享的流量特性

对于大多数新款安卓手机，可以实现在开启 WiFi 关闭流量的同时共享网络给 FreeBSD。这样就不会消耗流量了。

## 内核模块加载

首先需要加载相应的内核模块，以便系统能够识别 USB 网络共享设备（若默认未生效）。内核模块为系统提供了与特定硬件设备通信的能力。


### 一般安卓设备驱动

一般安卓（Android）设备（远程网络驱动接口规范，Remote Network Driver Interface Specification，RNDIS）：

```sh
# kldload if_urndis   # 加载 USB RNDIS 网络驱动
```

### 苹果设备驱动

苹果（Apple）iPhone/iPad 设备：

```sh
# kldload if_ipheth   # 加载 iPhone/iPad 以太网网络驱动
```

### 其他安卓设备驱动

其他安卓（Android）设备（通信设备类以太网控制模型/网络控制模型，Communication Device Class Ethernet Control Model/Network Control Model，CDC ECM/NCM）：

```sh
# kldload if_cdce     # 加载 USB CDC ECM/Ethernet 网络驱动
```



## 持久化驱动加载机制

若需要系统启动时自动加载上述模块，选择所需条目写入到 `/boot/loader.conf` 文件：

```ini
if_urndis_load="YES"    # 设置系统启动时自动加载 USB RNDIS 网络驱动
if_cdce_load="YES"      # 设置系统启动时自动加载 USB CDC ECM/Ethernet 网络驱动
if_ipheth_load="YES"    # 设置系统启动时自动加载 iPhone/iPad 以太网驱动
```



## 物理连接与网络共享启用

然后将 USB 数据线连接到 FreeBSD，打开手机的网络共享即可。

## 获取 IP 地址

为 `ue0`（通常是此接口，可通过 `ifconfig` 命令确认）接口通过 DHCP 获取 IP 地址：

```sh
# dhclient ue0
```

## 课后习题

1. 同时加载 `if_urndis` 和 `if_cdce` 模块，连接两部不同的 Android 手机，观察系统创建的网络接口命名规则，分析 FreeBSD 如何处理多个同类 USB 网络设备的资源分配。

2. 将 USB 网络共享接口 `ue0` 配置为静态 IP 并设置为默认网关，使用另一设备通过该接口上网。

3. 查阅 `if_urndis` 驱动源码，构建其最小可加载版本，分析为什么 USB 网络共享需要特定的内核模块支持而非作为用户空间程序实现。
