# 13.2 大模型本地部署

## llama.cpp

llama.cpp 致力于降低门槛，旨在通过最简化的配置，在各类硬件平台上实现极致的大模型推理性能。其主要使用 C/C++ 写就。

### 安装

- 使用 pkg 安装：

```sh
# pkg install llama-cpp
```

- 使用 Ports 安装：

```sh
# cd /usr/ports/misc/llama-cpp/
# make install clean
```

### 部署本地大模型

GGUF 是一种文件格式，存储了运行模型所需的信息。llama.cpp 要求模型存储为该格式。

[Hugging Face](https://huggingface.co/models?sort=trending&search=llama+gguf) 平台托管了大量适配 llama.cpp 的 GGUF 格式的大模型，可以手动搜索关键字“llama gguf”。

## Ollama

Ollama 是最快上手大型语言模型的方法，如 gpt-oss、Gemma 3、DeepSeek-R1、Qwen3 等。

Ollama 主要由 Go 语言和 C 语言编写。

### 安装

- 使用 pkg 安装：

```sh
# pkg install ollama
```

- 使用 Ports 安装：

```sh
# cd /usr/ports/misc/ollama/
# make install clean
```

## Claude Code

Claude Code 是一款 AI 编程助手和自主操作工具，可读取和理解整个代码库，编辑文件、运行命令，并与开发工具协作，支持在终端、IDE、桌面应用或浏览器中使用，帮助用户快速开发功能、修复漏洞和自动处理开发任务。

Claude Code 的源代码主要由 shell 和 Python 语言构成。

### 安装

- 使用 pkg 安装：

```sh
# pkg install claude-code
```

- 使用 Ports 安装：

```sh
# cd /usr/ports/misc/claude-code/
# make install clean
```

## GitHub Copilot CLI

GitHub Copilot CLI 是 GitHub 的闭源项目。GitHub Copilot CLI 将 AI 编程助手集成到命令行环境中，用户可以通过自然语言对话来编写、调试和理解代码，并与 GitHub 工作流程保持紧密配合。

### 安装

- 使用 pkg 安装：

```sh
# pkg install github-copilot-cli
```

- 使用 Ports 安装：

```sh
# cd /usr/ports/misc/github-copilot-cli/
# make install clean
```
