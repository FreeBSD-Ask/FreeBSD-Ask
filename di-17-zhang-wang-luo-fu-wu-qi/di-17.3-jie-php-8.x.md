# 第 17.3 节 PHP 8.X

> **注意**
>
> 不同的 PHP 模块之间是有冲突的，因此会造成 ports 编译失败，不要试图全选所有的 PHP 插件选项。因此推荐使用 pkg 进行安装。

```sh
# pkg install php82
```

> **注意**
>
> `82` 这个数字是会变动的。可以先 pkg search -o php 看看当前的版本是多少再安装。
