# 第 17.1 节 Apache

## 安装

```shell-session
# cd /usr/ports/www/apache24/ && make install clean
```

或

```shell-session
# pkg install apache24
```

## 启动服务

```shell-session
# 添加服务开机自启
# sysrc apache24_enable=YES

# 启动服务
# service apache24 start

# 查看状态
# service apache24 status
```

按理来说，apache 服务已经启动了，现在可以打开网址 `localhost` 看一下。
