# 第 29.3 节 安装 Awesome

awesome 是 X 的一个窗口管理器。任何布局都可以被动态地应用。为正在使用的应用程序和任务优化环境。可能不适用于 Wayland。

## 安装

```shell
#pkg install x11-wm/awesome
```
