# 18.2 树莓派 FreeBSD 使用

本章介绍树莓派安装 FreeBSD 系统后的基本配置和使用方法，包括 SSH 登录、时区设置、时间同步等常用操作。

## SSH 设置

本节介绍树莓派 FreeBSD 的 SSH（安全外壳协议，Secure Shell）登录方法。使用 XShell 登录树莓派，用户名和密码均为 `freebsd`。如需切换到 root 用户，请登录后执行 `su`，并输入 root 用户的密码。


## 调整时区

时区设置是系统配置的重要环节，正确的时区设置可以确保系统时间与本地时间一致。本节介绍如何设置系统时区。设置系统时区的方法如下：

```sh
# tzsetup
```

在设置时依次选择 `No` → `5 Asia` → `9 China` → `1 Beijing Time`，当提示“Does the abbreviation CST look reasonable?”时，选择 `Yes`。


## 时间设置

完成时区设置后，需要进一步配置系统时间同步机制，确保时间准确无误。树莓派 5 及之前的型号没有板载实时时钟（Real Time Clock，RTC）来保证时间的准确性，因此完全依赖网络时间协议（Network Time Protocol，NTP）服务进行时间校正。如果时间不准确，可能会影响多项服务的正常运行。

在 `/etc/rc.conf` 文件中添加以下配置：

```ini
ntpd_enable="YES"               # 启用 NTP 守护进程
ntpdate_enable="YES"            # 启用系统启动时使用 ntpdate 同步时间
ntpdate_program="ntpdate"       # 指定使用的时间同步程序
ntpdate_flags="0.cn.pool.ntp.org"   # 指定 NTP 服务器地址
```

启动 `ntpdate` 服务以同步系统时间：

```sh
# service ntpdate start
```

查看当前系统日期和时间：

```sh
# date
Tue Aug  1 16:45:55 CST 2023
```

### 主机名

主机名是系统在网络中的标识符，设置一个有意义的主机名有助于网络管理和设备识别。本节介绍如何设置主机名。在 `/etc/rc.conf` 文件中，可以修改主机名，例如 `hostname="generic"`，将 `generic` 替换为所需的主机名。


## CPU 频率

合理调整 CPU 频率可以在性能和功耗之间取得平衡，延长设备的使用寿命。本节介绍如何调整 CPU（中央处理器，Central Processing Unit）频率（范围为 600 MHz 到 1500 MHz）：

```sh
# sysrc powerd_enable="YES"     # 设置开机启用 powerd 动态调节 CPU 频率
# service powerd restart        # 重启 powerd 服务以应用配置
```

## 显示温度

监控 CPU 温度有助于及时发现散热问题，避免因过热导致的设备损坏或性能下降。本节介绍如何查看 CPU 温度信息。列出所有系统内核参数，并筛选包含温度的信息：

```sh
# sysctl -a | grep temperature
hw.cpufreq.temperature: 50598
dev.cpu.0.temperature: 51.0C
```

也可以查看各个 CPU 核心的温度信息：

```sh
# sysctl dev.cpu | grep temperature
dev.cpu.0.temperature: 50.1C
```

## 查看 CPU 频率

了解 CPU 当前的工作频率和可用频率等级，有助于评估系统性能状态。本节介绍如何查看 CPU 频率信息。显示 CPU 核心 0 的可用频率等级列表：

```sh
# sysctl dev.cpu.0.freq_levels
dev.cpu.0.freq_levels: 2000/-1 600/-1
```

## htop

htop 是一款功能强大的系统监控工具，能够直观地显示系统资源使用情况。本节介绍如何安装和配置 htop 系统监控工具。

- 使用 pkg 安装：

```sh
# pkg install htop
```

- 或者使用 Ports 安装：

```sh
# cd /usr/ports/sysutils/htop/ 
# make install clean
```

---

持久化设置（默认情况下，F10 保存后无法使用 `Ctrl + C`，必须选择 `quit` 才能保存设置；也可以通过文件进行指定）。

编辑 `~/.config/htop/htoprc` 文件，写入以下内容以显示 CPU 频率和温度：

```ini
show_cpu_frequency=1     # 启用 CPU 当前频率显示
show_cpu_temperature=1   # 启用 CPU 温度显示
```
