# 10.4 配置 OpenVPN

OpenVPN 已在 FreeBSD 的 Ports 提供。

## 安装 OpenVPN

- 使用 pkg 安装 OpenVPN：

```sh
# pkg install openvpn
```

- 或者使用 Ports 安装 OpenVPN：

```sh
# cd /usr/ports/security/openvpn/ 
# make install clean
```
