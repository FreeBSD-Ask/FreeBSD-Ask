# 8.4 OpenVPN

OpenVPN 在软件包中就有提供。

## 安装 OpenVPN

- 使用 pkg 安装：

```sh
# pkg install openvpn
```

- 或者使用 Ports 安装：

```
# cd /usr/ports/security/openvpn/ 
# make install clean
```
